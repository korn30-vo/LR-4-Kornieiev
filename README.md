# Лабораторна робота №4
# Виконав студент групи ПДМ-51 Корнєєв Володимир

Ця лабораторна робота присвячена дослідженню методів неконтрольованого навчання в машинному навчанні з використанням мови програмування Python та бібліотек NumPy, Matplotlib і Scikit-learn.

У ході роботи були реалізовані та проаналізовані основні алгоритми кластеризації: K-Means, Mean Shift та Affinity Propagation. Проведено експерименти на синтетичних даних, наборі Iris та фінансових даних.

У першому завданні виконано кластеризацію методом K-Means із побудовою меж кластерів та центроїдів. У другому завданні здійснено кластеризацію набору Iris із використанням зменшення розмірності (PCA). У третьому завданні застосовано алгоритм Mean Shift для автоматичного визначення кількості кластерів. У четвертому завданні використано Affinity Propagation для групування фінансових даних.
